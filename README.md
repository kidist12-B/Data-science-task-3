# Titanic Data Analysis & Custom Dataset

**Author:** Kidist Belay

---

## Project Overview

This repository contains two independent Python scripts that demonstrate data handling and analysis using the `pandas` library:

1. **Custom Dataset Construction** – Build a DataFrame from a dictionary with 5 columns, 15 rows, and a custom index.
2. **Titanic Survival Analysis** – Load, clean, explore, and derive actionable insights from the famous Titanic passenger dataset.

All code follows professional standards: clear variable naming, docstrings, modular functions, and comprehensive comments.

---

## Files

| File | Description |
|------|-------------|
| `part1_custom_dataset.py` | Creates an employee performance dataset. |
| `titanic_analysis.py` | Full Titanic analysis pipeline (cleaning, grouping, filtering, insights). |
| `Titanic-Dataset.csv` | Required Titanic data file (must be placed in the same directory). |
